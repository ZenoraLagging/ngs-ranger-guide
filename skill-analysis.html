Skill Descriptions

Weak Bullet

Main Class only Rifle only
Level 1 2 3 4 5
Damage 120%
Max Shots 3
Mark Duration 15s
Cooldown (after firing 3rd) 30s 25s 20s 15s 10s

Fires a bullet that increases damage dealt to the hit part.

Weak Bullet Reinforce

Main Class only
Level 1
Damage 125%
Mark Duration 15
Required Hit Count 100 hits

Parts hit by Weak Bullet that are hit a set number of times will take increased damage, and the effect duration of the skill will be extended.

Rifle Grenadier

Rifle only
Level 1
Trap Power 500%
Grenade Power 400
Cooldown 30s

When you have an Assault Rifle equipped, press and hold the Weapon Action button to add a grenade attack. Input a direction at the same time to throw the grenade, or input none to place the grenade at the current location.

Rifle Slide Shot Advance

Rifle only

With an Assault Rifle equipped, Weapon Actions will trigger an invulnerability period.

Spread Shot

Rifle only

Expend a gauge built up by Assault Rifle attacks to fire a special close range shotgun blast.

Spread Shot Quick Escape

Triggers an invulnerability period when you use Spread Shot. Inputting a direction will also extend the distance of the evasive action.

Launcher Group Charge

Launcher only
Level 1
Attack Count 1
Damage 90%

While using a Charged Normal Attack, Area of Effect increases by using a directional input forwards.

Cluster Bomb Advanced

Launcher only
Level 1
Cooldown 30s

Shortens the cooldown time between Weapon Actions while a Grenade Launcher is equipped.

Slow Landing Attack

Ranger weapons only

Slows fall speed when performing a Normal Attack with a Ranger weapon.

Slow Landing Charge

Ranger weapons only

Slows fall speed when charging a Ranger weapon Photon Art.
